## Learning suplimentary tools that work alongside
*RabbitMQ* is an open-source message broker that enables applications to communicate asynchronously, acting as a central hub for message queuing and routing

*Apache Kafka*  is an open-source distributed streaming platform used for building real-time data pipelines, streaming applications, and data integration at scale. It acts as a publish-subscribe messaging system that stores streams of records in a fault-tolerant and durable way, allowing producers to write data to topics and consumers to read from them. Key use cases include real-time analytics, log aggregation, and event-driven architectures.
